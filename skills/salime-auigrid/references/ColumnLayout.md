The AUIGrid Documentation describes how to configure column layouts, which must be declared as an array of objects, where each object represents a column. The core object for defining a column is `Column`.

The document details various renderers and edit renderers that can be used within column layouts:
*   **BarRenderer**: Displays a bar gauge in a cell.
*   **BTCalendarRenderer**: A Bootstrap calendar editor for date selection.
*   **ButtonHeaderRenderer**: Displays a button in the column header.
*   **ButtonRenderer**: Displays a button in a cell.
*   **CalendarRenderer**: A calendar editor for date selection.
*   **CheckBoxEditRenderer**: Displays a checkbox in a cell.
*   **CheckBoxHeaderRenderer**: Displays a checkbox in the column header.
*   **ComboBoxRenderer**: A combo box editor for selecting or inputting values.
*   **ConditionRenderer**: Dynamically assigns editors to column cells.
*   **CustomEditRenderer**: Allows developers to define custom cell editors.
*   **CustomRenderer**: Allows developers to define custom cell content.
*   **DropDownHeaderRenderer**: Displays a dropdown list in the column header.
*   **DropDownListRenderer**: Displays a dropdown list for single or multiple selections.
*   **IconRenderer**: Displays an icon in a cell.
*   **ImageRenderer**: Displays an image in a cell.
*   **InputEditRenderer**: A basic text input editor with validation capabilities.
*   **JQCalendarRenderer**: A jQuery UI-datepicker calendar editor for date selection.
*   **LinkRenderer**: Displays a hyperlink in a cell.
*   **MaskEditRenderer**: An editor that enforces a defined input pattern.
*   **NumberStepRenderer**: Displays a number stepping control for editing numerical values.
*   **RemoteListRenderer**: Fetches and displays lists from a remote source based on user input.
*   **SparkColumnRenderer**: Displays a spark column chart in a cell.
*   **SparkLineRenderer**: Displays a spark line chart in a cell.
*   **SparkWinLossRenderer**: Displays a spark win/loss chart in a cell.
*   **SwitchRenderer**: Displays a toggle switch in a cell.
*   **TemplateRenderer**: Renders raw HTML content in a cell.[1]
